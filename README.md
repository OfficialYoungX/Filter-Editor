## 简介
**Filter Editor**是一个基于css属性`css filter`的**滤镜效果合成器**。

特点：

1. 支持**拖拽**、**文件选取**、**剪贴板粘贴**上传图片。
2. 图片格式支持`png、jpeg、jpg、svg`等主流浏览器支持的图片格式。
3. 滤镜效果可以叠加。

当然，可以调节不同的参数，效果预览。
